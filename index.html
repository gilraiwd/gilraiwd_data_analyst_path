About Pivot Tracker

A personal tool for tracking a self-directed career transition — a 4-month checklist, portfolio project tracker, live job search, and progress dashboard. Everything you enter is stored only in your browser's local storage; nothing is uploaded anywhere. Built with AI assistance rather than hand-coded.

What's changed recently

  • Installable as an app Added a web app manifest and iOS meta tags — this page can now be installed to your home screen or desktop like a native app, using the browser's own install prompt (look for it in your browser's menu, or the install icon in the address bar on Chrome/Edge).
  • Fully editable "My Info" Job title, salary, target range, background summary, start date, and exam date all now drive the dashboard, header, and charts — nothing personal is hardcoded into the page anymore.
  • Skills-based live job search Replaced company-specific listings with a keyword-driven search (via a free public job board API) plus a live Indeed link. Remote-only, salary floor, and keyword filtering are all independently optional.
  • Adaptive skill radar The "Your Current Progress" line on the competency chart now moves as you complete the specific tasks that build each skill, instead of just showing a static target.
  • Pacing & exam countdown Dashboard cards comparing your actual progress to a time-based pace, plus a day countdown to your certification exam date.
  • Application pipeline Job applications now track through real stages (Targeted → Applied → Screening → Interview → Offer/Rejected) instead of a flat status label.
  • Backup tools Export/Import as JSON, a print/PDF view, and a nudge banner if it's been a while since your last backup.
